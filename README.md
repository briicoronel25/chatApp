# Chat App
This is a chat app assignment using socket.io and node.

## Before using

1. Do an NPM install
2. Download *socket.io, express, node and grunt*
3. Install and run node
4. Navigate to localhost:3030 and **START CHATTING!**

## This project include
node.js,
grunt,
package.json,
SASS.
